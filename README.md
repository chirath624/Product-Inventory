

This project is a modern Single Page Application (SPA) built with **React** and **Material UI**. It utilizes **Redux** for state management, **Redux Thunk** for asynchronous side effects (API calls), and **JSON Server** to act as a full fake REST API for prototyping.

## Tech Stack

* **Core:** [React](https://reactjs.org/) (Create React App )
* **UI Framework:** [Material UI (MUI)](https://mui.com/)
* **State Management:** [Redux](https://redux.js.org/) & [Redux Thunk](https://github.com/reduxjs/redux-thunk)
* **Backend Mock:** [JSON Server](https://dummyjson.com/docs)
* **HTTP Client:** Axios (recommended) or Fetch API
